Below is a brief summary of some of my recent projects:

## CHARM
[CHARM](https://github.com/robkavanagh/charm) is a framework I developed for characterising magnetic fields. It enables Bayesian inference of high-dimensioanl physical parameters from time-series data. This framework has been implemented into the workflows of international teams, leading to two publications in leading journals.

## MASER
[MASER](https://github.com/robkavanagh/maser) is a tool I developed for forecasting and analysing time series radio data of magnetic interactions between stars and planets. This enables teams to improve their data collection strategies without running computationally expensive simulations. I also recently published a [web interface](https://maser.robkavanagh.xyz/) for MASER, enabling usage without local install. MASER has been used to analyse time series by multiple international teams, resulting in two publications in leading journals.

## Decision science for high-stakes investments
The [Square Kilometre Array](www.skao.int/) is an upcoming €2 billion scientific instrument that will generate tens of TB per second from the night sky. Using my developed tools, I forecasted the expected ROI for different budget and infrastructure scenarios, directly informing stakeholders.
